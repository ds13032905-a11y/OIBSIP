# Calculator

## Description
A simple and responsive Calculator developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean and user-friendly interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Percentage Calculation
- Decimal Support
- Clear (C) Button
- Delete (DEL) Button
- Error Handling
- Invalid Input Handling

## Technologies Used
- HTML5
- CSS3
- JavaScript
